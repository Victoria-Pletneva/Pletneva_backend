## Описание предметной области
Приложение представляет собой REST API для управления сущностью **Car**.  
Это каталог автомобилей автосалона. Каждая машина имеет бренд, модель, год выпуска, цену и статус доступности.  
Все операции доступны только авторизованным пользователям (JWT-токен).

Сущность **Car** содержит:
- `id` (автоинкремент)
- `brand` (строка)
- `model` (строка)
- `year` (число ≥ 1900)
- `price` (число > 0)
- `available` (булево)

## Стек технологий
- **Backend**: Node.js + Express
- **База данных**: PostgreSQL 17
- **Авторизация**: JWT
- **Валидация**: Joi
- **Хеширование паролей**: bcryptjs
- **Подключение к БД**: pg + dotenv
